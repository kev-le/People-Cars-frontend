# People and Cars Frontend

### Installation steps:
1. run `npm install`
2. run `npm start`

The React web application is running on port `3001`

View the application at `http://localhost:3001`

Built with:
* react 16.13
* react-redux 7.2.0
* react-router 5.1.2
* redux-thunk 2.3.0
* redux 4.0.5
* connected-react-router 6.8.0
